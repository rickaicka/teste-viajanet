# Teste Viaja-Net

Para rodar o projeto, baixar o repositório,
acessar via cmd a pasta e rodar os seguintes comandos:

```
- npm install (para instalar as dependências do Node);
- ng server;

```
O projeto local rodará na url: http://localhost:4200/

Tecnologias usadas:

>- HTML5 e SASS;
>- Grid próprio;
>- Angular 6;
>- Typescript;
